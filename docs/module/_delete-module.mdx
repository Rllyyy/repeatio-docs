# Modul löschen

:::info In diesem Kapitel lernst du:

- Wie man ein Modul löscht

:::

1. Klick in der Modul Übersicht auf die drei Punkte (`⋮`) in dem Modul, welches gelöscht werden soll
2. Klick auf `Delete`

<img src={DeleteModule} width={550} style={{ display: "block", marginBottom: "10px", marginInline: "auto" }} />

3. Bestätige die Löschung, durch einen Klick auf `Delete Module`

:::caution Warnung
Das Löschen eines Moduls kann nicht rückgängig gemacht werden!
:::
