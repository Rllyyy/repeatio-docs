# Module exportieren

:::info In diesem Kapitel lernst du:

- Wie man ein Modul exportiert

:::

1. Klick in der Modulübersicht auf die drei Punkte (`⋮`) in dem Modul, welches exportiert werden soll
2. Klick auf `Export`

<img src={ExportModule} width={560} style={{ display: "block", marginBottom: "20px", marginInline: "auto" }} />

3. Abhängig vom verwendeten Browser öffnet sich ein Fenster, in dem der Speicherort festgelegt werden kann oder das Modul wird direkt in den Standard-Download-Speicherort heruntergeladen

:::tip Tipp
Im Bereich **[Settings](https://www.repeatio.de/settings)** können alle Module exportiert werden.
:::
