# Module exportieren

:::info In diesem Kapitel lernst du:

- Wie man ein Modul exportiert

:::

1. Klick in der Modulübersicht auf die drei Punkte (`⋮`) in dem Modul, das exportiert werden soll
2. Klick auf `Export`

<img src={ExportModule} width={560} style={{ display: "block", marginBottom: "20px", marginInline: "auto" }} />

3. Je nach Browser öffnet sich ein Fenster, um den Speicherort festzulegen. Oder das Modul wird in den Standardspeicherort heruntergeladen

:::tip Tipp
Im Bereich **[Settings](https://www.repeatio.de/settings)** können alle Module exportiert werden.
:::
