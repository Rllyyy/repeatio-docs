# Fragen-Editor

:::info In diesem Kapitel lernst du:

- Wie der Fragen-Editor aufgebaut ist
- Wie man Fragen formatiert
- Wie man Bilder und Formeln hinzufügt

:::

Mit dem Fragen-Editor können Fragen [erstellt](./questions/add-question) und [bearbeitet](./questions/edit-question) werden.  
Die mit einem roten Sternchen markierten Felder sind Pflichtfelder.  
Alle anderen Felder sind optional.

## Elemente

### ID

Die ID ist eine eindeutige Kennung für die Frage. Sie muss **einzigartig** in dem Modul sein und darf **keine Leer- oder Sonderzeichen** (#, /, !, ...) enthalten. Verwendet anstelle eines Leerzeichens einen Binde- oder Unterstrich.

### Titel

Formuliere den Titel so, dass er die Fragestellung klar und präzise wiedergibt. Im Titel kann MarkDown verwendetet werden, um den Text zu formatieren, Bilder einzufügen oder Tabellen zu erstellen.

### Points

Gib die Anzahl der Punkte an, die die Frage wert ist.

### Help

Im Help-Text kannst du zusätzliche Informationen zur Fragestellung oder zur Beantwortung der Frage hinzufügen. Der Help-Text wird unterhalb der Fragestellung angezeigt. Auch im Help-Text kann MarkDown verwendetet werden, um den Text zu formatieren, Bilder einzufügen oder Tabellen zu erstellen.

### Answer

Im Antwort-Text gibst du die korrekte Antwort auf die Frage ein. Der Antwort-Text wird unterhalb der Fragestellung und des Help-Textes angezeigt. Auch im Antwort-Text kann MarkDown verwendetet werden, um den Text zu formatieren, Bilder einzufügen oder Tabellen zu erstellen.

## Styling

Repeatio unterstützt die Gestaltung mit `Markdown` und `HTML` für die folgenden Elemente: **Titel**, **Hilfe** und **Antwortmöglichkeiten**.

### Bilder

Bilder können mit der Syntax `![Alt Text](www.link-to-image.com)` hinzufügen werden.

:::info Info
Der Alt Text wird angezeigt, wenn das Bild aus irgendeinem Grund nicht geladen werden kann, und dient auch der Barrierefreiheit für Sehbehinderte.
:::

  <table>
    <tr>
      <th>Markdown</th>
      <th>HTML</th>
      <th>Resultat</th>
    </tr>
    <tr>
      <td>

<!-- Test -->

```markdown
![md](link-to-image)
```

</td>
  <td>

```html
<img src="link-to-image" alt="md" />
```

</td>
      <td>
        <img src="https://upload.wikimedia.org/wikipedia/commons/d/d9/Markdown.png" alt="md" style={{height: "48px"}} />
      </td>
    </tr>
  </table>

### CSS

Benutze inline CSS in Kombination mit HTML, um ein Element zu stylen.

<table>
  <tr>
    <th>Input</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>

```markdown
Dieser Text ist nicht beinflusst von CSS

<p style="color: lightblue; font-weight: 700">Dieser Text ist blau</p>
```

</td>
    <td>
      <p>Dieser Text ist nicht beinflusst von CSS</p>
      <p style={{color: "lightblue", fontWeight: 700}}>Dieser Text ist blau</p>
    </td>
  </tr>
</table>

:::note Note  
Using html elements may overwrite the default styling!
:::

### Formeln (LaTeX)

Repeatio benutzt LaTeX, um Formeln zu rendern.  
Schreibe die Funktion zwischen zwei Dollarzeichen (`$`).

<table>
  <tr>
    <th>Input</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>

```markdown
$x =\frac{\sqrt{20}}{3\times(5-10)^2}$
```

</td>
    <td>

$x =\frac{\sqrt{20}}{3\times(5-10)^2}$

</td>

  </tr>
</table>

**[Hier](https://github.com/Rllyyy/repeatio/blob/main/.github/Docs/LaTeX.md)** ist eine detailierte Anleitung.
