# Was ist Repeatio?

<img src={Image} width={550} style={{ display: "block", marginInline: "auto", marginBottom: "40px" }} />

<p>

Repeatio ist eine **kostenlose Open Source Plattform**, die das Lernen für Prüfungen durch interaktive Lernmöglichkeiten vereinfacht.
Bereite dich durch ein gezieltes Training auf Prüfungen vor und verbessere deine Noten.

Diese Dokumentation führt durch die verschiedenen Funktionen von Repeatio.  
Um stattdessen direkt mit dem **Lernen anzufangen, besuche [repeatio.de](https://repeatio.de)**.

</p>

<p>Repeatio bereitet dich auf folgende Fragetypen vor:</p>

<ul>
  <li>
    <strong>Multiple Choice:</strong> Wähle die richtige Antwort aus.
  </li>
  <li>
    <strong>Multiple Response:</strong> Wähle mehrere richtige Antworten aus.
  </li>
  <li>
    <strong>Lückentext:</strong> Fülle die Lücken im Text aus.
  </li>
  <li>
    <strong>Extended Match:</strong> Ordne die Antwortenmöglichkeiten richtig zu.
  </li>
</ul>

:::caution Wichtige Hinweise

- Das Projekt befined sich in einem frühen Entwicklungsstadium, folglich können Fehler auftreten!
- Das **Leeren des Browser-Cache** (z.B. vor einer Prüfung) führt zum **Verlust der Daten**!
- Einige Features / UI-Elemente funktionieren noch nicht (z.B. Modulfortschritt, Statistiken, etc. pp.)!

:::

:::danger Achtung

Die Plattform ist nicht für die Durchführung von Prüfungen geeignet, da die Antworten unverschlüsselt lokal gespeichert werden. Die Plattform ist ausschließlich für das Vorbereiten auf Prüfungen gedacht.

:::

## Erste Schritte

Um mit dem Lernen zu beginnen, muss auf der [Plattform](https://repeatio.de) zunächst ein **[Modul erstellt](./module/create-module)** oder **[importiert](/module/import-module)** werden.  
Danach können in dem Modul **[Fragen hinzugefügt](./questions/add-question)** werden, die dann später in **[verschienden Modi](./practice-modes/modes)** interaktiv abgefragt werden können.

## Technische Vorraussetzungen

Repeatio unterstützt alle gängigen Endgeräte und Browser. Auf Apple Systemen kann es jedoch zu Anzeige Problemen kommen, da die Website nicht für Safari bzw. WebKit optimiert ist. Es wir empfohlen Firefox oder einen Chromium Browser (Google Chrome, Brave, Opera, Vivaldi, ...) zu nutzen. Achten Sie darauf ihren Browser auf dem neusten Stand zu halten.
Damit die Website funktioniert, muss JavaScript aktiviert sein.

## Technische Besonderheit

Alle Daten (Module, gespeicherte Fragen, Einstellungen, etc. pp.) werden ausschließlich lokal im Browser gespeichert. Es werden keine Daten an einen Server gesendet. Sollte der Cache des Browsers gelöscht werden, **gehen alle Daten verloren**! Die Daten können über die Export Funktion als JSON Datei gespeichert werden.
