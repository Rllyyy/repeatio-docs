# Was ist Repeatio?

<img src={Image} width={530} style={{ display: "block", marginInline: "auto", marginBottom: "40px" }} />

Repeatio ist eine **kostenlose Open Source Plattform**, die das Lernen für Prüfungen durch interaktive Lernmöglichkeiten vereinfacht.
Bereite dich durch ein gezieltes Training auf Prüfungen vor und verbessere deine Noten.

Diese Dokumentation führt durch die verschiedenen Funktionen von Repeatio.  
Um stattdessen direkt mit dem **Lernen anzufangen, besuche [repeatio.de](https://repeatio.de)**.

Mit Repeatio ist es möglich folgende Fragetypen zu erstellen und interaktiv zu lernen:

- **[Multiple Choice](/question-types#multiple-choice):** Wähle die richtige Antwort aus.
- **[Multiple Response](/question-types#multiple-response):** Wähle mehrere richtige Antworten aus.
- **[Lückentext](/question-types#lückentext-gap-text):** Fülle die Lücken im Text aus.
- **[Extended Match](/question-types#erweiterte-zuordnungsaufgabe-extended-match):** Ordne die Antwortenmöglichkeiten richtig zu.

:::caution Wichtige Hinweise

- Das **Leeren des Browser-Cache** (z.B. vor einer Prüfung) führt zum **Verlust der Daten**!
- Einige Features / UI-Elemente funktionieren noch nicht (z.B. Modulfortschritt, Statistiken, etc. pp.)!

:::

:::danger Achtung

Die Plattform ist nicht für die Durchführung von Prüfungen geeignet, da die Antworten unverschlüsselt lokal gespeichert werden. Die Plattform ist ausschließlich für das Vorbereiten auf Prüfungen gedacht.

:::

## Erste Schritte

Um mit dem Lernen zu beginnen, muss auf der [Plattform](https://repeatio.de) zunächst ein **[Modul erstellt](./module/add-module#modul-erstellen)** oder **[importiert](/module/add-module#modul-importieren)** werden. Danach können in dem Modul **[Fragen hinzugefügt](./questions/add-question)** werden, die dann später in **[verschienden Modi](./modes)** interaktiv abgefragt werden können.

## Technische Vorraussetzungen

Repeatio unterstützt alle gängigen Endgeräte und Browser. Auf Apple Systemen kann es jedoch zu Anzeige Problemen kommen, da die Website nicht für Safari bzw. WebKit optimiert ist. Empfohlen wird die Verwendung von Firefox oder eines Chromium-Browsers (Google Chrome, Brave, Opera, Vivaldi, ...). Für eine fehlerfreie Nutzung wird empfohlen, immer die neuste Version des Browsers zu verwenden. JavaScript muss aktiviert sein, damit die Website korrekt funktioniert.

## Technische Besonderheit

Alle Daten (Module, gespeicherte Fragen, Einstellungen, etc. pp.) werden ausschließlich lokal im Browser gespeichert. Es werden keine Daten an einen Server gesendet. Sollte der Cache des Browsers gelöscht werden, **gehen alle Daten verloren**! Die Daten können über die Export Funktion als JSON Datei gespeichert werden.
