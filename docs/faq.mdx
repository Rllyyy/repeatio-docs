# FAQ

<details open={false}>
<summary><b>Wo werden die Dateien gespeichert?</b></summary>

Alle Dateien werden lokal auf dem Gerät gespeichert. Grund dafür sind Bedenken hinsichtlich des Datenschutzes, des Urheberrechts und möglicher Serverkosten.

</details>

<details open={false}>
<summary><b>Kann man die Website zum Schreiben echter Prüfungen verwenden?</b></summary>

Nein, die Website ist nicht für die Verwendung in echten Prüfungen gedacht. Sie ist lediglich als Lernhilfe konzipiert. Fragen und Antworten liegen unverschlüsselt auf dem Gerät und können somit von dem Nutzer eingesehen werden.

</details>

<details open={false}>
<summary><b>Wie kann ich Dateien sichern oder auf ein anderes Gerät übertragen?</b></summary>

Einzelne Module können in der Module Overview durch einen Klick auf die drei Punkte rechts neben dem Modul heruntergeladen werden. Auf dem anderen Gerät können die Module dann durch einen Klick auf Add Module und Import Module hinzugefügt werden. Alle Module, gemerkten Fragen und Einstellungen können in den Einstellungen kompakt heruntergeladen werden.

</details>

<details open={false}>
<summary><b>Wie kann ich Fehler melden oder neue Funktionen vorschlagen?</b></summary>

Fehler und Feature-Wünsche können über das GitHub-Repository (https://github.com/Rllyyy/repeatio/issues) gemeldet oder per E-Mail an contact@repeatio.de geschickt werden. Achte darauf, die Beschreibung so detailliert wie möglich zu gestalten. Verwende dazu Bilder oder genaue Fehlermeldungen.

</details>

<details open={false}>
<summary><b>Ist das Projekt wirklich kostenlos?</b></summary>

Ja, das Projekt ist 100% kostenlos und es gibt keine versteckten Kosten. Das Projekt ist Open Source. Das bedeutet, dass der Quellcode für jedermann zugänglich ist.

</details>

<details open={false}>
<summary><b>Wie kann ich Fragen gestalten?</b></summary>

Fragen können mit Hilfe von MarkDown und CSS gestaltet werden. **[Hier](./question-editor#styling)** erhältst du mehr Informationen dazu.

</details>

<details open={false}>
<summary><b>Kann man beim Projekt mitwirken?</b></summary>

Ja, das Projekt ist Open Source und kann auf GitHub (https://github.com/Rllyyy/repeatio) eingesehen und bearbeitet werden. Achte darauf, im bestehenden Ecosystem zu bleiben (React, Vite, TypeScript, TailwindCSS und Cypress).

</details>
