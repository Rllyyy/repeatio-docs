# Frage löschen

:::info In diesem Kapitel lernst du:

- Wie man eine Frage löscht

:::

1. Navigiere zur Frage, die gelöscht werden soll
2. Klick unten rechts auf das Löschen Icon

<img src={DeleteQuestion} width={460} style={{ display: "block", marginBottom: "10px", marginInline: "auto" }} />

:::info Tipp

Auf einem mobilen Endgerät, musst du eventuell zuerst unten rechts die Navigationsleiste (:arrow_up:) öffnen, um das Löschen Icon zu sehen.

:::

3. Bestätige das Löschen der Frage
