# Frage bearbeiten

:::info In diesem Kapitel lernst du:

- Wie man eine Frage bearbeitet

:::

1. Navigiere zu der Frage, die bearbetet werden soll
2. Klick auf das Bearbeiten Icon in einer Frage

<img src={EditQuestion} width={460} style={{ display: "block", marginBottom: "10px", marginInline: "auto" }} />

3. Fülle die notwendigen Felder aus des [Fragen-Editors](../question-editor.mdx) aus
4. Klick auf `Update`

:::info Info
Auf mobile Endgeräten muss eventuell zunächst die Navigationsleiste ausgeklappt werden.
:::
