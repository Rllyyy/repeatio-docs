# Styling

Repeatio unterstützt die Gestaltung mit `Markdown` und `HTML` für die folgenden Elemente: **Titel**, **Hilfe** und **Antwortmöglichkeiten**.

:::tip Tipp
Versuch immer die elemente zunächst mit Markdown zu stylen und benutzt HTML nur, wenn Markdown nicht den gewünschten Effekt erzielt. Dies ist zum Beispiel der Fall, wenn man CSS verwenden möchte, das Standardstyling überschreiben will oder mehrere Zeilenumbrüche haben möchtest.
:::

## Zeilenumbrüche

<p>
  Zeilenumbrüche können in Markdown durch drücken der <code>Enter</code> Taste erreicht werden. Um einen Zeilenumbruch
  in HTML zu erreichen, muss <code>&lt;br /&gt;</code> verwendet werden.
</p>

<table>
  <tr>
    <th>Typ</th>
    <th>Markdown</th>
    <th>HTML</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>Einzelner Zeilenumbruch</td>
    <td>
      Oberer Text <br /> Neue Zeile
    </td>
    <td>
      Oberer Text <code>&lt;br /&gt;</code> Neue Zeile
    </td>
    <td>
      Oberer Text <br />
      Neue Zeile
    </td>
  </tr>
  <tr>
    <td>Mehrere Zeilenumbrüche</td>
    <td>
      Text in mehrere<code>&ensp;</code>
      <sup>
        <a href='#footnote-line-break-1'>[1]</a>
      </sup>
      <br />
      Zeilen aufgeteilt
    </td>
    <td>
      Text in mehrere <code>&lt;br /&gt;&lt;br /&gt;</code>Zeilen aufgeteilt
    </td>
    <td>
      Text in mehrere <br /> <br /> Zeilen aufgeteilt
    </td>
  </tr>
</table>

<p id='footnote-line-break-1'>[1]: Notice how there are two blank spaces!</p>

## Markdown Formatierung ignorieren

Benutze Rückstriche, um die Markdown Formatierung zu ignorieren.

<table>
  <tr>
    <th>Input</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>

```markdown
Dieser \*Text ist nicht beinflusst\* von Markdown
```

  </td>
    <td>
      <p>Dieser *Text ist nicht beinflusst* von Markdown</p>
    </td>
  </tr>
</table>

## Listen

Es gibt zwei Arten von Listen: **Ungeordnete** und **Geordnete** Listen.

  <table>
    <tr>
      <th>Typ</th>
      <th>Markdown</th>
      <th>HTML</th>
      <th>Resultat</th>
    </tr>
    <tr>
      <td>Ungeordnete Listen</td>
      <td>

```markdown
Text...

- Erstes ungeordnetes Element
- Ein weiteres Element
- Letztes Element

Text außerhalb der Liste
```

</td>
      <td>

```html
Text...
<ul>
  <li>Erstes ungeordnetes Element</li>
  <li>Ein weiteres Element</li>
  <li>Letztes Element</li>
</ul>
Text außerhalb der Liste
```

  </td>
      <td>
        <p>Text...</p>
        <ul>
          <li>Erstes ungeordnetes Element</li>
          <li>Ein weiteres Element</li>
          <li>Letztes Element</li>
        </ul>
        <p>Text außerhalb der Liste</p>
      </td>
    </tr>
    <tr>
      <td>Geordnete Listen</td>
<td>

```markdown
Text...

1. Erstes geordnetes Element
2. Ein weiteres Element
3. Letztes Element

Text außerhalb der Liste
```

</td>
      <td>

```html
Text...
<ol>
  <li>Erstes geordnetes Element</li>
  <li>Ein weiteres Element</li>
  <li>Letztes Element</li>
</ol>
Text außerhalb der Liste
```

  </td>
      <td>
        <p>Text...</p>
        <ol>
          <li>Erstes geordnetes Element</li>
          <li>Ein weiteres Element</li>
          <li>Letztes ElementLetztes Element</li>
        </ol>
        <p>Text außerhalb der Liste</p>
      </td>  
    </tr>
  </table>

:::caution Achtung
Um eine Liste in Markdown zu beenden, ist es es notwending eine leere Zeile zwischen der Liste und dem folgenden Inhalt oder zwei Leerzeichen am Ende des letzten Listenelements einzufügen.
:::

:::info Info
Ungeordnete Listen in Markdown können Sterne (<code>\*</code>), Minus-Symbole (<code>-</code>) oder Plus-Symbole (<code>+</code>) verwenden.
:::

:::tip Tipp
Benutze die folgendene Syntax unterhalb der Überschrift, um Elemente in einer Markdown Tabelle auszurichten:

- Links (`| :------- |`) ist optional, da es identisch zu `| ------- |` ist
- Mitte (`| :-------: |`)
- Rechts (`| -------: |`)

:::

## Bilder

Bilder kann man mit der Syntax `![Alt Text](www.link-to-image.com)` hinzufügen.

:::info Info
Der Alt Text wird angezeigt, wenn das Bild aus irgendeinem Grund nicht geladen werden kann, und dient auch der Barrierefreiheit für Sehbehinderte.
:::

## CSS

Benutze inline CSS in Kombination mit HTML, um ein Element zu stylen.

<table>
  <tr>
    <th>Input</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>

```markdown
Dieser Text ist nicht beinflusst von CSS

<p style="color: lightblue; font-weight: 700">Ich bin blau</p>
```

</td>
    <td>
      <p style={{color: "lightblue", fontWeight: 700}}>Ich bin blau</p>
    </td>
  </tr>
</table>

:::note Note  
Using html elements may overwrite the default styling!
:::

## Formeln (LaTeX)

Repeatio benutzt LaTeX, um Formeln zu rendern.  
Schreibe die Funktion zwischen zwei Dollarzeichen (`$`).

<table>
  <tr>
    <th>Input</th>
    <th>Resultat</th>
  </tr>
  <tr>
    <td>

```markdown
$x =\frac{\sqrt{20}}{3\times(5-10)^2}$
```

</td>
    <td>

$x =\frac{\sqrt{20}}{3\times(5-10)^2}$

</td>

  </tr>
</table>

**[Hier](https://github.com/Rllyyy/repeatio/blob/main/.github/Docs/LaTeX.md)** ist eine detailierte Anleitung.
